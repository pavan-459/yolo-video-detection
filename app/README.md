# NavaSoft App — Next.js

This is the Next.js full-stack application for the NavaSoft CRUD assignment.

For full setup instructions, tech stack details, API reference, and architecture overview see the **[root README](../README.md)**.

## Quick Start

```bash
# From this directory (app/)
node server.js
```

Runs on `http://localhost:3000`. Requires MongoDB and the Flask ML server to also be running — see the root README for full instructions.
